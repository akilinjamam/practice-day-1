Inline Block Element

What is Inline block element?

The element which have the both quality of inline and block. the can set by defult vartically and horizontally.they can be given height and width. But the block element can not set side by side as like as inlie element rather than one after one vartically and they allow margin padding all around. on the other hand the inline element can not set one after one from top to bottom. they can set side by side only.

By defult inline block elements

Button: the element button tag is a by defult inline-block element.When the button binding applies to a button element, the element is expected to render as an 'inline-block' box rendered as a button whose contents are the contents of the element.

Select: select is also inline block element.The Select element is used to create a drop-down list. The select element is most often used in a form, to collect user input. The name attribute is needed to reference the form data after the form is submitted (if you omit the name attribute, no data from the drop-down list will be submitted).

what is HTML semantic Elements?

Semantic HTML elements are those that clearly describe their meaning in a human- and machine-readable way. this Elements are all considered semantic because they accurately describe the purpose of the element and the type of content that is inside them.
